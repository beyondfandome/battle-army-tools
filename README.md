# Battle Army Tools v0.1.2

Army-scale battle helpers for Foundry VTT.

## Included in v0.1.2

- Automatic movement watcher
  - cumulative movement tracking
  - terrain movement costs
  - routed units blocked from manual movement
  - routed-pool teleport bypass for Resolve Combat
  - occupied-square blocking for active battle units
- Enhanced hover tooltip
  - colour-coded status and HP
  - sectioned layout for core stats, turn state, command, and terrain
  - attack / defence / ranged defence / range terrain bonus breakdowns
  - movement terrain penalty / bonus display
  - ammo and Can Volley display
  - command tokens and formation info
- Turn HUD
  - round
  - current side
  - current phase
  - current side commander command tokens
- Custom HP bars

## Install / update

Use this manifest URL in Foundry:

https://raw.githubusercontent.com/beyondfandome/battle-army-tools/main/module.json

For GitHub release assets, upload this ZIP as:

battle-army-tools-v0.1.2.zip

## Notes

Resolve Combat is still intentionally left as a macro for now. Combat rules are changing quickly, and player-safe GM socket combat should be handled in a later module version.
